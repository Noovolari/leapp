    How to create a IAM Federated Role Access Method for AWS

    This section describes how to setup IAM Federated Role Access Method for AWS. It is applicable to different Identity Provider; for the sake of this tutorial, G Suite represents the Identity Provider.

    To correctly setup this Access Method you've to go through the following steps:

    1. G Suite Federation Setup;
    2. AWS Federation Setup;
    3. Enable role federated access;
    4. Assign role to G Suite Principal;
    5. Setup in Leapp.
